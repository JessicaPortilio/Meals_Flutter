# MEALS
Nesse aplicativo de refeições, temos uma lista com vários tipos de refeições para que o usuário possa escolher. 

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805219-b763fb00-e2f1-11ea-9c3a-39c33827905a.png"/>
<br>

Ao clicar na opção desejável, ele vai para outra página onde terá vários alimentos daquele tipo que foi escolhido. 

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805224-b9c65500-e2f1-11ea-975a-00577c7391fd.png"/>
<br>

Escolhendo a refeição desejável, você é direcionado para outra página onde tem os ingredientes e os passos para fazer aquela receita.

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805242-bd59dc00-e2f1-11ea-932e-23c3fe13884f.png"/>
<br>

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805265-c64aad80-e2f1-11ea-8334-d1da714aa7fc.png"/>
<br>

Você pode marcar como favorito suas receitas que você gostar e encontrará elas nas categorias de favoritos na parte inferior do aplicativo.

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805275-c9459e00-e2f1-11ea-899a-cfe91b0decd6.png"/>
<br>

Também temos uma TabBar, onde você consegue ir para página de refeições (página inicial), como ir para página de configurações. 

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805281-cba7f800-e2f1-11ea-882d-782a5fc61643.png"/>
<br>

Entrando na parte de configurações, você pode escolher os alimentos que você não quer que apareça para você. 

<br>
<img height="650" src="https://user-images.githubusercontent.com/47065396/90805288-cd71bb80-e2f1-11ea-87ca-7f0cc665f9fb.png"/>
